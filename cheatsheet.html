Cheat sheet

This page is built to be a resource for you as you're learning how to write HTML, CSS, and JavaScript. Working in descending order, there are code snippets and accompanying examples of what the code produces. You are allowed to copy/paste as needed. While it will take some practice to understand how these pieces fit together, and we'll do other things in class to learn their syntax, you are welcome to use this page as a "cheat sheet."

HTML

Hypertext Markup Language (HTML) is the backbone of any website. On this page, you'll be able to find every component we talk about for our projects. Also, to make it easier, there's a list included that will take you directly to the component you're looking for.

Paragraphs of Text

This is the easiest way of getting text into your project. Use this tag whenever you want to just include some basic text.


<p>
    Hello there! I'm some text.
</p>

Hello there! I'm some text.

Headings

Headings offer a bit more styling. These decrease in size depending on your CSS. They work, in decreasing size, from 1-6.


<h1>One</h1>
<h2>Two</h2>
<h3>Three</h3>
<h4>Four</h4>
<h5>Five</h5>
<h6>Six</h6>

One

Two

Three

Four

Five
Six

Unordered Lists

An unordered list is just a series of bullets - there's no real indexing of the items in the list; thus, it's "unordered."


<ul>
    <li>I'm an item!</li>
    <li>Me, too!</li>
</ul>
  • I'm an item!
  • Me, too!

Ordered Lists

On the other hand, ordered lists are those who have ranked items (i.e., an order to them).


<ol>
    <li>First</li>
    <li>Second</li>
    <li>Third</li>
</ol>
  1. First
  2. Second
  3. Third

Anchor Tags

Want to find a way to get away? Anchor tags will link to other pages or other sections of the same page. "href" is an attribute that stands for hyperlink reference and is the address you're taking the user to.


<a href="https://www.google.com">Click me to go to Google!</a>
<a href="#cheat-header">Beam me back up!</a>
Click me to go to Google! Beam me back up!

CSS

Cascading Style Sheets (CSS) make our page look clean and give it a definite style. While HTML makes up the skeleton of our page, CSS makes it look good! The syntax of CSS differs from that of HTML, but is pretty quick to pick up once you know the names of different properties.

Background Color

The quickest way of telling if something has been affected by CSS is to change its background color. Here, we give a block a red background.


<div class="block">
    I have a red background!
</div>

<style>
    .block {
        background: red;
    }
</style>
I have a red background!

Text Color

To change the color of the text, simply modify the "color" property of an element.


    <div class="block">
        I have a red background AND white text!
    </div>
    
    <style>
        .block {
            background: red;
            color: white;
        }
    </style>
I have a red background AND white text!

Padding

Things can get really squished sometimes, so we use "padding" to modify the amount of space inside an element.


        <div class="block">
            I have a red background AND white text!
        </div>
        
        <style>
            .block {
                background: red;
                color: white;
                padding: 10px;
            }
        </style>
I have a red background AND white text!

Margin

Alternatively, maybe we want to change how much space is between an element and the others around it. This would be adjusting the "margin", or the space around an element.


            <div class="block">
                I have a red background AND white text!
            </div>
            
            <style>
                .block {
                    background: red;
                    color: white;
                    padding: 10px;
                }
            </style>
I have a red background AND white text!

Hover

The style of elements can change based on certain events. For example, when a user hovers over the following element, the background and text colors will change.


                <div class="block">
                    I have a red background AND white text!
                </div>
                
                <style>
                    .block {
                        background: red;
                        color: white;
                        padding: 10px;
                    }

                    .block:hover {
                        background: darkred;
                        color: yellow;
                        padding: 10px;
                        cursor: pointer;
                    }
                </style>
I have a red background AND white text!
